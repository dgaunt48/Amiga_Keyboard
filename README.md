# Amiga_Keyboard

Amiga RP2350B Keyboard Tester.

![VGA Output](Pics/Output.jpg?raw=true "VGA Output")

Tested using my RP2350b 40GPIO board.

![VGA Output](Pics/Keyboard.jpg?raw=true "VGA Output")

GPIO 9 = KEYBOARD_CLOCK  (Black)

GPIO 10 = KEYBOARD_DATA  (Brown)

GPIO 11 = KEYBOARD_RESET (Red)

5v0 = (Orange)

GND = (Green)
